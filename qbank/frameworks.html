QWhen would you use LangGraph over plain orchestration?Advancedtechnical deep-diveSeniorfreq: emergingBoth
Model answer

LangGraph models an agent as a graph of nodes and edges with explicit shared state. Reach for it when you need:

  • Explicit state passed between steps.
  • Conditional routing (different next step based on results).
  • Checkpointing — retry/resume a single step, not the whole run.
  • Human-in-the-loop approval at certain points.
Strong senior-level answer

A senior uses it for control and observability but notes you can achieve the same with disciplined plain Python — the framework is a convenience, not the architecture.

Likely follow-ups
  • What does the checkpointer give you?
Red-flag answer pattern
Using it just because it's popular, with no reason.

QLangChain vs LlamaIndex — how do you frame the difference?Intermediatetechnical deep-diveSeniorfreq: mediumBoth
Model answer
  • LangChain — broad orchestration toolkit (chains, agents, integrations).
  • LlamaIndex — centered on data indexing and retrieval for RAG.
  • Both are conveniences layered over the same underlying primitives (LLM calls, embeddings, vector search).
Strong senior-level answer

A senior picks by fit, keeps the abstraction thin, and is willing to drop to the raw API when the framework hides too much.

Likely follow-ups
  • When do frameworks hurt you?
Red-flag answer pattern
Treating a framework as the system instead of a tool.
QWhat's the risk of heavy framework abstraction in production?Advancedsystem designSeniorfreq: emergingBoth
Model answer

Heavy abstractions can:

  • Hide the actual prompts and control flow, making debugging hard.
  • Make cost and latency harder to see and control.
  • Break on version upgrades.
Strong senior-level answer

A senior keeps a clean seam so the framework is replaceable, and traces exactly what gets sent to the model regardless of the framework.

Likely follow-ups
  • How do you see what the framework actually sends the model?
Red-flag answer pattern
Having no idea what prompt the framework is actually sending.
